# Drone And Turrets
## Gameplay
### Moving keys
- W - up
- S - down
- D - right
- A - left
- left shift - fly up
- left ctrl - fly down
### Goal the game 
Destroy all turrets at the level. 
### Restore Actors
- Box with Ammo
- Heal Wrench
### Enemy
Turret.
They can shoot in you!

### Main Character
You can shoot and fly :)
